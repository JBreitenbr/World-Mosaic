# World Mosaic  
**Von einer muffigen Werkhalle zur globalen Ästhetik der Tagesstrukturen**

## Die unerwartete Entstehung

Dieses Projekt begann nicht in einem Atelier, nicht in einer Bibliothek,  
nicht einmal an einem schönen Ort.
Seine Entstehungsgeschichte wurde vielmehr von einer **muffigen Werkhalle** inspiriert.

Dort, ausgerechnet dort, stellte sich die Frage:
**Wie sieht eigentlich *Tagesstruktur* aus?**
Zunächst lokal gedacht,  
dann philosophisch,  
dann atmosphärisch —  
und dann plötzlich **global**.  

## Idee & Konzept

**World Mosaic** sammelt  
*Tagesrhythmen aus aller Welt*:  
ruhige, klare Miniaturen aus Städten, Landschaften, Häfen, Hügeln, Treppen, Gassen,  
geschrieben im Stil nüchterner, kleiner Porträts.

Kein Drama.  
Keine Tourismusästhetik.  
Keine Überhöhung.

Nur Alltag als präzises kulturelles Mikroklima.

Jede Seite zeigt:

- **eine Person**  
- **einen Ort**  
- **mehrere Zeitpunkte**  
- **eine Atmosphäre**  
- **ein reduziertes lokales Design** (Farben, Texturen, Icons)

So entsteht ein **globales Mosaik**,  
gebaut aus den unspektakulären Momenten,  
die am ehrlichsten von einer Kultur erzählen.

---

## Ästhetischer Ansatz

- hell, warm, minimalistisch  
- keine Postkarten  
- keine Klischees  
- kleine grafische Elemente  
- leichte, abstrahierte Texturen  
- jeder Ort hat eine **eigene Farbgrammatik**  
- jedes Land bekommt **sein eigenes CSS**  
- und alles bleibt dennoch **ruhig**, **präzise**, **literarisch**

Kurz gesagt:  
**Tagesstruktur als Weltarchitektur.**

---

## Warum dieses Projekt existiert

Weil eine muffige Werkhalle aus Versehen  
zum katalytischen Muse-Ort wurde.

Weil Alltag weltweit der beste Geschichtenerzähler ist.  
Weil man über Zeitstempel erstaunlich weit reisen kann.  
Weil HTML & CSS sich perfekt eignen,  
um Atmosphären zu miniaturisieren.

Und weil es plötzlich Sinn machte,  
einen globalen Rhythmus sichtbar zu machen - ohne Lärm, ohne Pathos,  
einfach durch Beobachtung.
